# Instalação
`git clone https://github.com/dev-moises-lima/consultas-frontend-aprimorado.git` => clona o repositório.<br>
`npm install` => instala as dependências do projeto.<br>
Crie um arquivo .env na raiz do seu projeto. Copie e cole o conteúdo do arquivo .env.example nesse arquivo. É necesário ainda colocar as informações do pusher nas respectivas variáveis de ambiente.<br>
`npm run dev` => inicia o servidor de desenvolvimento.
